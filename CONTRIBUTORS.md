# Contributors

This repository does not accept external contributions.

The cost of producing code has become cheaper than the cost of reviewing code from other contributors. We prefer to solve problems ourselves.

Instead of submitting a pull request, you are encouraged to create an issue describing the problem or feature request:

- **Bug reports**: Open an issue with steps to reproduce, expected behavior, and actual behavior.
- **Feature requests**: Open an issue describing what you would like to see and why.

We review issues regularly and will address them as priorities allow.
