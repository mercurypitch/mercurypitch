# User Experience Guidelines

## Practical and Multi-Device experience

- Allow microphone enabling even if playback is running (as user might forget)
- Depends whether its smart to do so, or not,
- if its not smart, allow during pause
- Keyboard shortcuts
  - Play/Pause/Continue, Stop
  - Go from beginning
  - Mode switches (Repeat/Practice)
    - Remove once as it is not needed (mode active by default)

- Reorganization of layouts in session mode and focus mode as well as small vs big screen mode!

- Default settings, sensitivity PRESETS
- Quiet room (no background noise preset, studio quality quiet room)
- Some noise (at home preset)
- High noise environment (outside preset)

- Ask user before mic on or on welcome screen!
- welcome screen is yet to be implemented (see: )
- By Default Some noise preset

- Adjustability - finer granularity in settings, along with preset mode switch

- Allow to hide:
  - All gridlines (especially focus mode)
  - Note reference / key info
  - Stats / Run data
  - Live pitch tracker (left and bottom)
    - maybe gonna be useful but hide by default

- In focus mode:
  - Add up/down arrows for switching speed (0.25, 0.5, 0.75, 1.0x etc) of the playback.
